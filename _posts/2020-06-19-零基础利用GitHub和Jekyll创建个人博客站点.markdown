# 前言
	若干年前，还在读书那会，QQ空间是最流行的个人站点，不管主流还是非主流，都有装饰一番的欲望，朋友之间还会互踩，作为文学爱好者的我基本上每周都写一篇日志。
	而工作之后，QQ基本被微信所替代，人也懒了，也逐渐没有了倾述的欲望。
	
	此番提起兴致利用github创建个人博客，一是想践行一个短期的计划，另外也是为了能在心血来潮之时能有个地方记录一二。
